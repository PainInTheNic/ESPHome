# ESPHome
YAML code for ESPHome devices to be used with Home Assistant.

These are template files that require you to complete one or both of the following:
  1.  Variables defined as *Substitutions* at the beginning of the file. This is where you define your device's name, description, as well as any necessary information for expanded functionality features.
  2.  There may be configuration details required for some expanded functionality features to work properly. This is most commonly providing a UUID.
